# 📖 About

This repository contains implementations of the **Stack** data structure in **C**. It covers both **Array** and **Linked List** implementations along with fundamental stack operations and practical applications.

The repository is designed for students learning **Data Structures & Algorithms**, preparing for coding interviews, or practicing university assignments.
